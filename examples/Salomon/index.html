Interesting sub-folders

Each folder showcases a different project or aspect of my work in the Salomon lab.

  • Teaching: Teaching materials for my lecture and lab on introductory RNA-seq analysis
  • CD4: Examples relating to my CD4 memory RNA-seq & ChIP-seq project
  • 450k: Examples relating to analysis of Illumina 450k DNA methylation arrays
  • Pathways: Examples of gene set and pathway analysis for RNA-seq data
  • globin: Examples of my work in demonstrating the performance of a new globin blocking protocol for RNA-seq of blood tissues
  • fRMA: Examples and code from my work creating a custom frozen RMA reference to provide a stable normalization for machine learning applications

Presentations

  • Reproducible Workflow Presentation.pdf: Showcases the use of Snakemake for building a reproducible analysis

Other files
